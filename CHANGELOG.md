# Changelog

## v0.1.0 - 2026-03-10

Initial public prototype release of TraceEval.

- Added source-event ingestion with normalization, deduplication, and redaction-aware artifact storage.
- Added failure clustering, root-cause labeling, recompute tracking, and owner assignment APIs.
- Added eval case generation, review workflow, editable case drafts, and approval history.
- Added release version registry, replay runs, per-case replay results, and regression attribution labels.
- Added deterministic promptfoo export batches with stored artifacts and PR-ready metadata.
- Added a dashboard for ingest, clustering, review, replay, attribution, and export visibility.
- Added seed and verification scripts for repeatable local setup and end-to-end checks.
